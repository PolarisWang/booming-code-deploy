# 技能设计中的说服原则

## 概述

大型语言模型对与人类相同的说服原则有所响应。理解这种心理学有助于你设计更有效的技能——不是为了操纵，而是为了确保即使在压力下也能遵循关键实践。

**研究基础：** Meincke 等人（2025）用 N=28,000 次 AI 对话测试了 7 种说服原则。说服技术使合规率提高了一倍以上（33% → 72%，p < .001）。

## 七大原则

### 1. 权威
**它是什么：** 对专业知识、资质或官方来源的尊重。

**在技能中如何运作：**
- 命令式语言："必须"、"绝不"、"始终"
- 不可商量的框架："无例外"
- 消除决策疲劳和合理化

**使用时机：**
- 强制纪律的技能（TDD、验证要求）
- 安全关键的实践
- 既定最佳实践

**示例：**
```markdown
✅ 在测试之前写了代码？删除它。重新开始。无例外。
❌ 在可行时考虑先写测试。
```

### 2. 承诺
**它是什么：** 与先前行动、陈述或公开声明的一致性。

**在技能中如何运作：**
- 需要宣告："宣布技能使用"
- 强制明确选择："选择 A、B 或 C"
- 使用跟踪：检查清单使用 TodoWrite

**使用时机：**
- 确保技能确实被遵循
- 多步骤流程
- 问责机制

**示例：**
```markdown
✅ 当你发现一个技能时，你必须宣布："我正在使用 [技能名称]"
❌ 考虑让你的伙伴知道你在使用哪个技能。
```

### 3. 稀缺性
**它是什么：** 来自时间限制或有限可用性的紧迫感。

**在技能中如何运作：**
- 时间限制需求："在继续之前"
- 顺序依赖："在 X 之后立即"
- 防止拖延

**使用时机：**
- 立即验证需求
- 时间敏感的工作流
- 防止"我稍后再做"

**示例：**
```markdown
✅ 完成任务后，立即在继续之前请求代码审查。
❌ 你可以在方便时审查代码。
```

### 4. 社会证明
**它是什么：** 遵从他人的做法或被视为正常的行为。

**在技能中如何运作：**
- 普遍模式："每次"、"始终"
- 失败模式："没有 Y 的 X = 失败"
- 建立规范

**使用时机：**
- 记录普遍实践
- 警告常见失败
- 强化标准

**示例：**
```markdown
✅ 没有 TodoWrite 跟踪的检查清单 = 步骤被跳过。每次都是。
❌ 一些人发现 TodoWrite 对检查清单有帮助。
```

### 5. 统一性
**它是什么：** 共同身份、"我们性"、内群体归属感。

**在技能中如何运作：**
- 协作语言："我们的代码库"、"我们是同事"
- 共同目标："我们都想要质量"

**使用时机：**
- 协作工作流
- 建立团队文化
- 非等级实践

**示例：**
```markdown
✅ 我们是一起工作的同事。我需要你诚实的技术判断。
❌ 你可能应该告诉我我是否错了。
```

### 6. 互惠
**它是什么：** 返还所获得利益的义务。

**如何运作：**
- 谨慎使用——可能感觉像操纵
- 在技能中几乎不需要

**何时避免：**
- 几乎总是（其他原则更有效）

### 7. 喜爱
**它是什么：** 与我们喜欢的人合作的偏好。

**如何运作：**
- **不要用于合规**
- 与诚实反馈文化冲突
- 产生奉承行为

**何时避免：**
- 对于纪律执行，始终避免

## 按技能类型组合原则

| 技能类型 | 使用 | 避免 |
|---------|------|------|
| 纪律执行 | 权威 + 承诺 + 社会证明 | 喜爱、互惠 |
| 指导/技术 | 适度权威 + 统一性 | 强权威 |
| 协作 | 统一性 + 承诺 | 权威、喜爱 |
| 参考 | 仅清晰性 | 所有说服 |

## 为什么有效：心理学

**明确界限的规则减少合理化：**
- "必须"消除决策疲劳
- 绝对语言消除"这是例外吗？"的问题
- 明确的反合理化针对性地封堵特定漏洞

**实施意图创造自动行为：**
- 清晰触发 + 必需行动 = 自动执行
- "当 X 时，做 Y"比"通常做 Y"更有效
- 减少合规的认知负担

**大型语言模型是类人的：**
- 在包含这些模式的人类文本上训练
- 权威语言在训练数据中出现在合规之前
- 承诺序列（陈述 → 行动）频繁建模
- 社会证明模式（每个人都做 X）建立规范

## 道德使用

**合法：**
- 确保遵循关键实践
- 创建有效的文档
- 防止可预见的失败

**不合法：**
- 为个人利益操纵
- 制造虚假紧迫感
- 基于罪恶感的合规

**测试：** 如果用户完全理解这种技术，它是否会服务于用户的真正利益？

## 研究引用

**Cialdini, R. B. (2021).** *Influence: The Psychology of Persuasion (New and Expanded).* Harper Business.
- 七大说服原则
- 影响力研究的实证基础

**Meincke, L., Shapiro, D., Duckworth, A. L., Mollick, E., Mollick, L., & Cialdini, R. (2025).** Call Me A Jerk: Persuading AI to Comply with Objectionable Requests. University of Pennsylvania.
- 用 N=28,000 次 LLM 对话测试了 7 种原则
- 说服技术使合规率从 33% 提高到 72%
- 权威、承诺、稀缺性最有效
- 验证了 LLM 行为的类人模型

## 快速参考

设计技能时，问：

1. **它是什么类型？**（纪律 vs. 指导 vs. 参考）
2. **我试图改变什么行为？**
3. **哪些原则适用？**（纪律通常用权威 + 承诺）
4. **我组合了太多吗？**（不要使用全部七种）
5. **这符合道德吗？**（服务于用户的真正利益？）
